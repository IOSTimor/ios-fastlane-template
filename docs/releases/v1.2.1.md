# v1.2.1 Release Notes Draft

Patch release focused on stabilizing Homebrew installation.

## Fixed

- Homebrew formula now downloads the installer script from the tagged `v1.2.1` release instead of the moving `main` branch
- `ios-fastlane-template` now prints the correct CLI usage when installed from Homebrew
- Tap-based Homebrew installs are no longer vulnerable to raw GitHub branch cache mismatch

## Installation

Tap and install:

```bash
brew tap IOSTimor/homebrew-tap https://github.com/IOSTimor/homebrew-tap
brew install IOSTimor/homebrew-tap/ios-fastlane-template
```

Then run:

```bash
ios-fastlane-template /path/to/your/ios-project
```
