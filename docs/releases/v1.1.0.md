# v1.1.0 Release Notes Draft

Reusable iOS fastlane release template for projects that already manage signing in Xcode.

## Highlights

- Added `testflight_only`, `submit_review`, and `precheck_assets` lanes
- Improved installer auto-detection for project, workspace, scheme, and app name
- Added safer installer behavior that refuses to overwrite existing fastlane setups unless forced
- Added `curl | bash` installer flow
- Added Homebrew formula support
- Added `en-US` and `zh-Hans` starter metadata content
- Added screenshot guidance for both locales
- Added GitHub Actions validation for lanes and installer flows
- Added `SECURITY.md`, changelog, contributing guidance, and issue/PR templates

## Installation

Install with `curl`:

```bash
curl -fsSL https://raw.githubusercontent.com/IOSTimor/ios-fastlane-template/main/scripts/install.sh | bash -s -- /path/to/your/ios-project
```

Install with Homebrew:

```bash
brew install --formula https://raw.githubusercontent.com/IOSTimor/ios-fastlane-template/main/Formula/ios-fastlane-template.rb
ios-fastlane-template /path/to/your/ios-project
```

## Upgrade Notes

- If you already have a `fastlane/` directory, the installer now refuses to overwrite it by default
- Metadata templates now include optional fields such as subtitle, promotional text, marketing URL, and app name
- Screenshot locale README files are guidance only and are ignored by screenshot prechecks
